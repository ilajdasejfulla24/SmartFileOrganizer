# Smart File Organizer

## Description
A Python OOP application that automatically organizes files into folders according to file extensions.

## Features

- Object-Oriented Design
- Automatic File Classification
- Automatic Folder Creation
- Logging System
- Undo Function
- Tkinter GUI
- Start/Stop Organizing
- Error Handling

## Technologies

- Python
- Tkinter
- shutil
- json
- logging
- os

## Installation

Clone the repository:

git clone <repository-url>

## Run

python main.py
